## New Application architecture

We're building an animal library of breeds. We'll probably do something more with it down the line, but for now we're trying to get this up and running.

### Debug
I've written this PoC in Go. To run it, open this folder in vscode and run in 'debug'. You'll be able to browse via a web browser to the site using one of the following.
```
https://localhost:8080/animal/dog
https://localhost:8080/animal/cat
https://localhost:8080/animal/cow
```
You can also use `curl` or `Postman`, both viable options if you prefer CLI tools. I recommend you get comfortable with these.
